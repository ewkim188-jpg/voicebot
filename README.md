## ğŸŒ English Introduction

**VoiceBot** is an intelligent voice assistant application that recognizes user speech, generates AI responses, and plays them back via audio.

### ğŸš€ Key Features
- **Real-time Speech-to-Text (STT)**: High-accuracy voice transcription using OpenAI's **Whisper** model.
- **Advanced AI Models**: Powered by **GPT-4o**, GPT-4 Turbo, and GPT-3.5 Turbo for intelligent and natural conversations.
- **Natural Text-to-Speech (TTS)**: Seamless audio playback of AI responses using **gTTS (Google Text-to-Speech)**.
- **Web-based UI**: Simple and responsive interface built with **Streamlit**.
- **Chat History**: Visual tracking of the ongoing conversation.
